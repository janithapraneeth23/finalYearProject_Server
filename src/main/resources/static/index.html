ABSTRACT

As mobile computing and cloud computing working together to provide new technologies and capabilities to end user, clouds are overloading with data-intensive and compute intensive processes. Video streaming is a such application trending in the technology world. Video based content has been an important media for communication and entertainment. With clouds potential, editing and processing video streams can be easily done. Even so while watching a real time video stream, a single buffering situation can be distracting to most users. Real time stream processing not just latency sensitive but also computer intensive and memory intensive task. Standalone clouds cannot handle such latency. Cloudlets are important and new architectural components in mobile and cloud architecture, representing the middle tier of a mobile device - cloudlet - cloud hierarchy. Using cloudlets, this research trying to resolve that live video stream processing which are computer & memory intensive can be done smoothly. And produce new genre of video streaming applications. Different types of video editing algorithms can run on the same video stream and create different streams as per user request
